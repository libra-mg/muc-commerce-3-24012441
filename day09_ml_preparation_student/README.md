## 用自己的话回答

- 什么是特征，什么是标签：特征是模型做判断的依据，在这里是用户的各项属性数据；标签是我们要预测的真实答案，在这里是用户最终是否流失。
- 为什么要保留测试集：测试集用来检验模型对从未见过的新数据能否正确判断，防止模型只是死记硬背训练集而缺乏泛化能力。
- 为什么83%准确率仍可能没有用：因为数据中未流失用户本就占83%，即使什么都不学、全部预测为未流失，也能达到这个准确率。关键要看召回率——如果真实流失用户一个都没找出来，模型就毫无业务价值。
